## Приложение обработки заказов клиентов

Обрабатывает заказы от клиентов из различных источников:
 - Email
 - FTP
 - API
 - Telegram

 Заказы имеют различные форматы:
 - xlsx
 - xml
 - csv
 - json

 Приложение использует следующие технологии:
 - Net 6
 - ASP.Net Core 6
 - Entity Framework Core 6
 - SQLite
